# 蝴蝶精灵图扩展指南

## 一、当前资源

黄色蝴蝶的 PNG 文件，16x80 像素，包含 5 个 16x16 的素材，组成一个侧视图蝴蝶翅膀煽动的循环动画

- **单帧尺寸**: 16 × 16 像素

- **当前问题**: 所有精灵带有黑色描边，且只有一个方向

## 二、目标

1. 从当前的单一视角，拓展到 5 个视角，使用完全相同的风格和大小

| 视角 | 角度 | 描述                         |
| ---- | ---- | ---------------------------- |
| 正面 | 0°   | 蝴蝶正对观察者               |
| 前侧 | 45°  | 蝴蝶向左前方                 |
| 侧面 | 90°  | 蝴蝶完全朝左侧（与现有相同） |
| 后侧 | 135° | 蝴蝶向左后方                 |
| 背面 | 180° | 蝴蝶背对观察者               |

确保所有视角的蝴蝶身体中心对齐

2. 去掉黑色描边

将黑色替换为合适自然的颜色，这里需要参考到蝴蝶自身的颜色，不要简单把当前描边变成透明，因为要保证蝴蝶的大小是和现在一致的。

## 三、交付内容

Aseprite 工程文件

输出的图片素材

对黄色的蝴蝶，输出单张动画素材，大小为 80x80 像素

解释：

输出 5x5 个素材，每个素材的像素和当前保持一致，即 16x16，拼接就是 80x80

第一行：正面视角的动画
第二行开始依次类推，参考上方表格中的顺序

必须保持不同视角之间动画的一致性，即，每一列来看，蝴蝶自身是一个状态，只是观察角度不同
